### 👤 Mengenai Saya
Saya seorang peminat teknologi yang gemar meneroka dunia digital dan fizikal. Apabila saya tidak menghadap skrin atau membelek gajet terkini, saya biasanya berada di jalanan, merakam momen-momen menarik menggunakan kamera 360°, atau merancang pengembaraan seterusnya.

* 🔭 **Fokus Sekarang:** Membangunkan projek web (Eleventy SSG) & mengoptimumkan skrip automasi.
* 📸 **Minat Mendalam:** Fotografi digital, videografi (360° content), dan penghasilan aset stok kreatif.
* 🏍️ **Jiwa Kembara:** Penggemar *long-distance motorcycle touring* dan peminat sejarah/kebudayaan Asia.
* 💬 **Tanya Saya Tentang:** Android/Pixel setup, video rendering, atau rute *touring* lintas sempadan terbaik.
