### 👋 Coretan Sang Pengembara Biasa
Salam hormat! Saya hanyalah seorang peminat jalanan yang kini sedang gigih bertatih meneroka lebuh raya digital. Kalau di alam nyata, saya seronok memulas *throttle* merentas sempadan, tapi di alam maya ni, saya masih seorang pelajar yang banyak merangkak, mencuba, dan adakalanya tersadung dengan *error* kod. Ilmu koding saya tak seberapa, banyak bergantung pada tunjuk ajar *master* di luar sana and hasil *trial & error*.

Sama ada menyusun laluan kembara ke sempadan Thailand, menelaah sejarah tamadun kuno, atau sekadar mencari penyesuaian (*color grading*) terbaik untuk video, saya sentiasa seronok belajar benda baru. Tegur-tegurlah kalau terserempak, baik di jalanan mahupun di GitHub ini!
